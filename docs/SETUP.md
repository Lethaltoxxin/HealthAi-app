## Groq API Key Setup (FREE)

1. Go to https://console.groq.com
2. Sign up / log in and create a new API key (completely free)
3. Create a file called `.env` in the project root
4. Add this line: `VITE_GROQ_API_KEY=paste_your_key_here`
5. Restart the dev server: `npm run dev`

## Features

The AI Health Assistant supports:
- **Health Q&A** — Ask any health question and get evidence-based answers
- **Prescription Analysis** — Upload a prescription photo for drug analysis, dosage, interactions
- **Report Analysis** — Upload lab results for clear explanations of each value
- **Symptom Checking** — Describe symptoms for possible conditions with urgency triage
- **Incognito Mode** — Private sessions that aren't saved
- **Explainability** — View confidence % and source citations for each response
